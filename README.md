# SOLEHAUS — Premium Sneaker Store (Demo)

A fully responsive, single-file premium sneaker e-commerce storefront built as a client demo.

## Features
- Modern dark premium UI with volt accent
- Product grid with category filters (Lifestyle / Running / Basketball / Limited)
- Quick-view product modal with size selector
- Working cart drawer (add / remove / quantity / subtotal)
- Toast notifications, live countdown timer, animated marquees
- Fully mobile responsive

## Tech
- Pure HTML + CSS + vanilla JavaScript (no build step, no dependencies)
- Google Fonts (Archivo + Inter)
